# Project Setup Guide

## Overview

This guide will help you set up and run both the client and server sides of your project.

---

## Client

### Technologies Used:

- **React**
- **TypeScript**
- **Vite**

### Setup Instructions:

1. **Navigate to the Client Directory:**
   ```sh
   cd ./Client
   ```
2. **Install Dependencies:**
   ```sh
   npm install
   ```
3. **Start the Development Server:**
   ```sh
   npm run dev
   ```

---

## Server

### Technologies Used:

- **Node.js**
- **Express**
- **TypeScript**

### Setup Instructions:

1. **Navigate to the Server Directory:**
   ```sh
   cd ./server
   ```
2. **Install Dependencies:**
   ```sh
   npm install
   ```
3. **Start the Development Server:**
   ```sh
   npm run dev
   ```

---

## Additional Notes

- Ensure you have Node.js and npm installed on your machine.
- For any issues during setup or running the servers, refer to the respective documentation of the technologies used.
